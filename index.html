Installation

NPM

npm install tilted.js

or

CDN

https://cdn.jsdelivr.net/npm/tilted.js/dist/tilted.js

Properties

intensity: controlles the intensity (default: 12)
onHover: controlles if tilt happens only on hover (default: false)
disableX: enables/disables tilt on the X axis (default: false)
disableY: controlles the intensity (default: false)
invert: tilt will be inverted (default: false)
